# 🛒 Amazon Product Review Sentiment Analyzer

## 🌐 Live Demo

**Live Application:**  
https://amazon-feedback-analysis.streamlit.app/

## 📌 Project Overview

This project uses **Machine Learning** and **Natural Language Processing (NLP)** to analyze Amazon product reviews and predict whether a review expresses a **Positive**, **Neutral**, or **Negative** sentiment.

The application is built using **Python**, **Scikit-learn**, and **Streamlit**, providing users with a simple web interface for real-time sentiment prediction.

---

## 🚀 Features

- 📊 Data Loading and Preprocessing
- 📈 Exploratory Data Analysis (EDA)
- ☁️ Word Cloud Visualization
- 😊 Sentiment Classification
- 🔤 TF-IDF Feature Extraction

## ▶️ How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/chiluka-lakshmipriya/Amazon-Product-Review-Analysis.git
```

### 2. Navigate to the project folder

```bash
cd Amazon-Product-Review-Analysis
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit application

```bash
streamlit run app.py
```

### 5. Open the application

Visit:

```
http://localhost:8501
```

Enter an Amazon product review and click **Predict Sentiment** to view the result.
