# Amazon Product Review Sentiment Analysis

## 📌 Project Overview

This project uses **Machine Learning** and **Natural Language Processing (NLP)** to analyze Amazon product reviews and predict whether a review is **Positive**, **Neutral**, or **Negative**.

## 🚀 Features

* Data Loading and Preprocessing
* Exploratory Data Analysis (EDA)
* Word Cloud Visualization
* Sentiment Classification
* TF-IDF Feature Extraction
* Logistic Regression Model
* Streamlit Web Application
* Real-time Review Prediction

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* WordCloud
* Streamlit
* Joblib

## 📊 Model Performance

* **Algorithm:** Logistic Regression
* **Feature Extraction:** TF-IDF Vectorizer
* **Accuracy:** **86.58%**

## 📂 Project Files

* `Amazon_Review_Analysis.ipynb` – Jupyter Notebook containing the complete project
* `app.py` – Streamlit web application
* `model.pkl` – Trained Machine Learning model
* `tfidf.pkl` – Saved TF-IDF vectorizer

> **Note:** The original `Reviews.csv` dataset is not included in this repository because it exceeds GitHub's file size limit.

## ▶️ How to Run the Project

1. Install the required Python packages:

```bash
pip install -r requirements.txt
```

2. Start the Streamlit application:

```bash
streamlit run app.py
```

3. Open the browser and visit:

```text
http://localhost:8501
```

4. Enter an Amazon product review and click **Predict** to view the sentiment.
